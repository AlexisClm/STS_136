Je vous recommande au préalable de lire le fichier "ReadMe.txt" du dossier "x64" afin de connaître les contrôles ainsi que le but du jeu

- Pour jouer au jeu : allez dans le dossier "x64/Release" et cliquez sur "ProjetSMFL.exe"
- Pour voir le code source : allez dans le dossier "src" et cliquez sur les fichiers se terminant par ".h" et ".c"

Vous pouvez également modifier les valeurs de certaines données en accédant au fichier "GameParameters.txt" (exemple : remplacer la valeur au-dessous de #PlayerPropMax par 90 pour augmenter le nombre de propulsions maximum)
